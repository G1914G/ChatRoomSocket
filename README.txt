Tested in Intelij
Step 1: Open the project.
Step 2: Open src -> Server. In the Server package run the Server by right clicking on Server.java and click 'Run Server.main()'.
Step 3: Open src -> Client. In the Client package run the Client by right clicking on Client.java and click 'Run Client.main()'.
Step 4: Repeat step 3 to make multiple clients.
Step 5: Test the application. Go to another chat by clicking a chat name in the left list.
Step 6: Always close the client app by clicking the red cross in the right corner of the app.
Step 7: Close the server in Intelij.

Note:
The server prints lines in the console to show what is happening on the server side.
